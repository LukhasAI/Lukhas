## üîß Technical Implementation

### DNS Configuration
```
lukhas.ai     A     your-server-ip
www           CNAME  lukhas.ai
api           A      api-server-ip
```

### Nginx Configuration
```nginx
server {
    server_name lukhas.ai www.lukhas.ai;
    root /var/www/lukhas.ai;

    location /api {
        proxy_pass http://localhost:8000;
    }
}
```

### Deployment Script
```bash
#!/bin/bash
# Deploy to all domains
for domain in ai dev io store cloud id eu us xyz team; do
    echo "Deploying to lukhas.$domain"
    rsync -avz ./build/ root@lukhas.$domain:/var/www/
done
```
