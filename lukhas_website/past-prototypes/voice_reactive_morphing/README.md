## üöÄ Quick Start

### Prerequisites
- Modern web browser with WebGL support
- Microphone access for voice features
- Internet connection for AI integration

### Installation
1. Clone the repository:
```bash
git clone https://github.com/lukhas-systems/proteus.git
cd proteus/final-demo
```

2. Open `index.html` in your browser:
```bash
open index.html
```

3. Configure your API keys in the settings panel (‚öôÔ∏è)

### Basic Usage
1. **Enable Voice Interface**: Click "Start Voice" to begin voice processing
2. **Enable LUKHlukhasS AGI**: Click "Enable LUKHlukhasS" to activate AI integration
3. **Speak or Type**: Interact with the system through voice or chat
4. **Observe Transformations**: Watch shapes morph in response to your input
