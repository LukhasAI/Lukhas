## Brand Usage Rules

### MΛTRIZ Display Form
- **Usage**: Logos, hero sections, marketing materials, wordmarks
- **Context**: Display/promotional content only
- **Typography**: Greek Lambda (U+039B) required
- **Accessibility**: Must include `aria-label="Matriz"`

```html
<h1 aria-label="Matriz">MΛTRIZ</h1>
```

### Matriz Plain Form
- **Usage**: Body copy, SEO, metadata, alt text, documentation
- **Context**: All text content and accessibility contexts
- **SEO**: Primary term for search optimization

### URL/Slug Format
- **Correct**: `/matriz`, `matriz-documentation.pdf`
- **Incorrect**: `/mλtriz`, `MATADA-legacy.pdf`

## Vendor Language (Neutral)
- Use "uses [Provider] APIs" or "integrates with [Provider] services"
- Avoid "powered by", "partnered with", or implied endorsements
- Example: "MATRIZ uses OpenAI APIs for language processing" (not "powered by OpenAI")
- Λ display is for wordmarks only; use plain "Lukhas" in body text/metadata
