## 1. Romantic Poetry Techniques

### 1.1 John Keats' Synesthesia

**Definition**: Metaphors and descriptions that simultaneously appeal to two or more senses, producing effects that are both hallucinatory and vivid.

**Key Examples**:
- "Full of dewey wine" (visual + tactile + taste)
- "Melodious plot of beechen green" (sound + visual)
- "Sunburnt mirth" (visual + emotional)
- "Musk roses" described as having fragrance so thick it hangs like incense

**Techniques**:
- **Sensory Overload**: Layer multiple sense impressions simultaneously
- **Gustatory Imagery**: Use taste associations to create warm, positive images
- **Olfactory Differentiation**: Distinguish by fragrance when vision is obscured
- **Temporal Blending**: Mix present sensations with anticipated future experiences

**Application to LUKHAS**:
```
‚ùå "AI processes information quickly"
‚úÖ "Consciousness distills meaning like dewfall crystallizing overnight, each insight tasting of discovery's bitter-sweet clarity"
```

**Core Theory**: Distinction between Primary Imagination (universal human faculty), Secondary Imagination (poetic/creative power), and Fancy (mechanical recombination).

### 8.2 Novelty Score Calculation

**Target**: ‚â•0.8 novelty score across all poetic hooks within 30-day semantic windows

**Formula**:
```
Novelty Score = 1 - (Semantic Similarity / Max Possible Similarity)

Where:
- Semantic Similarity = cosine similarity of embedding vectors
- Threshold for "repetitive" = >0.8 similarity
- Threshold for "novel" = <0.2 similarity
- Target range = 0.15-0.3 similarity (novelty 0.7-0.85)
```

**Practical Guidelines**:

1. **Vocabulary Rotation**: Cycle through 8 families every ~4 weeks
2. **Synonym Variation**: Never use same metaphor twice within 10 pages
3. **Structural Variation**: Alternate sentence structures (active/passive, simple/complex)
4. **Perspective Shifts**: Vary viewpoint (consciousness as agent vs object)

**Example Analysis**:
```
Hook 1: "Consciousness roots spread through digital soil"
Hook 2: "Awareness roots grow through neural ground"
Similarity: ~0.75 (TOO HIGH - essentially same metaphor)
Novelty: 0.25 (BELOW TARGET)

Fix:
Hook 2: "Understanding phototropic, bending toward truth's light"
Similarity: ~0.25 (ACCEPTABLE - related domain, different metaphor)
Novelty: 0.75 (ABOVE TARGET)
```
