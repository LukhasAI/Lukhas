### Core APIs

#### Consciousness Interaction API
**Endpoint**: `https://api.lukhas.ai/v1/consciousness`

Interact directly with LUKHAS consciousness systems through our primary API. Features include:

- **Consciousness Queries**: Ask questions and receive conscious responses
- **Context Preservation**: Maintain conversation context across interactions
- **Ethical Filtering**: All responses filtered through our Guardian System
- **Multi-Modal Support**: Text, image, and structured data interaction

```javascript
// Example: Basic consciousness interaction
const response = await lukhas.consciousness.query({
  message: "Help me understand the ethical implications of this decision",
  context: "business-strategy",
  ethicsLevel: "thorough"
});
```

**Endpoint**: `https://api.lukhas.ai/v1/guardian`

### Code Examples

#### Basic Interaction
```python
import lukhas

# Initialize consciousness connection
consciousness = lukhas.Consciousness(api_key="your-api-key")

# Ask a question with ethical consideration
response = await consciousness.query(
    message="What are the ethical implications of this business decision?",
    context={
        "industry": "healthcare",
        "stakeholders": ["patients", "healthcare workers", "investors"],
        "timeline": "next 5 years"
    },
    ethics_level="thorough"
)

print(f"Analysis: {response.analysis}")
print(f"Ethical Score: {response.ethics_score}")
print(f"Recommendations: {response.recommendations}")
```

#### Memory Integration
```javascript
const lukhas = require('@lukhas/consciousness-sdk');

// Create a persistent consciousness session
const session = await lukhas.createSession({
  userId: 'user123',
  contextPreservation: true,
  ethicsLevel: 'standard'
});

// Interact with memory preservation
const response1 = await session.query("I'm working on a new product design");
const response2 = await session.query("What ethical considerations should I keep in mind?");

// The second query will remember the product design context
```

#### Guardian Integration
```python
# Analyze a decision for ethical implications
decision = {
    "action": "launch_ai_product",
    "description": "New AI assistant for healthcare workers",
    "stakeholders": ["doctors", "patients", "hospital_administrators"],

ethics_analysis = await lukhas.guardian.analyze_decision(decision)
```
