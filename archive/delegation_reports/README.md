Delegation reports generated from Ruff JSON output.

Files:
- `ruff-master-report.json`  — per-file issues with short suggestions
- `ruff-by-code.json`       — per-rule grouping with counts and top files
- `ruff-summary.json`       — top-level summary and totals

These files were generated by `scripts/tools/generate_ruff_delegation_reports.py`.

Use these artifacts to delegate lint fixes (per-file or per-rule).
